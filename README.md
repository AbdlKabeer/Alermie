# Alermie

Alermie is a modern mobile application built with React Native and Expo. It provides a robust alarm clock and time management experience.

## Features
- Set multiple alarms with customizable times and ringtones.
- Simple, user-friendly interface.
- Cross-platform support (iOS and Android).

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
